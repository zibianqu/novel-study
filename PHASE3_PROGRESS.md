## 🏗️ 技术架构

```
知识图谱系统
├─ Neo4j 数据库层
│   ├─ 节点类型
│   │   ├─ Character (人物)
│   │   ├─ Location (地点)
│   │   ├─ Event (事件)
│   │   ├─ Item (物品)
│   │   └─ Concept (概念)
│   └─ 关系类型
│       ├─ KNOWS (认识)
│       ├─ LOCATED_AT (位于)
│       ├─ HAPPENS_AT (发生于)
│       ├─ OWNS (拥有)
│       └─ CAUSES (导致)
│
├─ 图谱构建引擎
│   ├─ EntityExtractor (实体抽取)
│   ├─ RelationExtractor (关系抽取)
│   ├─ GraphBuilder (图谱构建)
│   └─ ConsistencyChecker (一致性检查)
│
├─ 查询服务层
│   ├─ GraphQueryService
│   ├─ PathFinder (路径查询)
│   ├─ GraphReasoner (图推理)
│   └─ GraphStats (统计分析)
│
└─ 应用层
    ├─ WritingAssistant (写作助手)
    ├─ ConsistencyValidator (一致性验证)
    └─ PlotAnalyzer (剧情分析)
```
